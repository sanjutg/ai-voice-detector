# AI Voice Detector

A FastAPI-based application that detects whether a given audio sample
is AI-generated or human.

## Live Demo
https://ai-voice-detector-h26q.onrender.com/docs

## Tech Stack
- FastAPI
- Python
- Uvicorn
- Render

## How it Works
Audio is sent as Base64, analyzed for pitch and spectral patterns,
and classified as AI or Human.
